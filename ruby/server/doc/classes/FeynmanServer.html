Module FeynmanServer
In: lib/feynman_server.rb

Synopsis

The Feynman Server is a web server that has the specific capability of executing simulations built using the Feyman Simulation Framework.

Usage

ruby feynman_server.rb [start|stop] [options]

-p, —port
- Which port to bind to
-d, —daemonize
- Run daemonized in the background
-l, —log
- Where to write the log messages
-P, —pid
- Where to write the PID file
-a, —address ADDR
- Address to bind to (default: localhost)
-s, —storage
- Storage location (default: /usr/local/feynman/storage)
-C, —config PATH
- Use a config file to specify server options
-f, —force
- Issue KILL instead of TERM (specific to stop)

Examples

  ruby feynman_server.rb start -C /usr/local/webservers/feynman/config/
